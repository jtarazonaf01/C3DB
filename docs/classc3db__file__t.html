Detailed Description

Low-level file wrapper used by C3DB to isolate disk access.

The class intentionally has no database semantics: no log, no cache and no record validation. Higher layers decide how records are interpreted and which partial writes are safe.

When both header_size and rec_size are zero, the wrapper behaves as a raw byte stream starting at base_addr.
