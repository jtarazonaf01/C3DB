273 /*
274 * Cache storage model:
275 *
276 * cache_memory_ owns the only block that stores cached entries. Entries are
277 * addressed by an absolute entry number in this block:
278 *
279 * seq: [0, seq_capacity_)
280 * hist: [seq_capacity_, seq_capacity_ + hist_capacity())
281 * write: [seq_capacity_ + hist_capacity(), total_capacity_)
282 */

312 /*
313 * Sequential-access history:
314 *
